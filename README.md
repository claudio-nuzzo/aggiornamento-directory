# Istituto Stradivari - Gestione Passaggio d'Anno

Strumento web statico (HTML5/JavaScript) per l'automazione delle procedure di fine anno scolastico su Google Workspace. 
Distribuito in modo sicuro tramite Cloudflare Pages.

## 🔒 Sicurezza e Privacy (GDPR)
L'applicazione è al 100% Client-Side. I file Excel e CSV caricati dagli utenti **non vengono inviati a nessun server internet**. Tutta l'elaborazione e il confronto dei dati avvengono esclusivamente all'interno della memoria del browser del collaboratore.

## 🛠️ Tecnologie Utilizzate
- **Tailwind CSS**: Per l'interfaccia utente scannabile e reattiva.
- **SheetJS (xlsx.js)**: Per la lettura e il parsing dei file Excel direttamente nel browser.
